# Laravel Trivy Sample

This is a sample application demonstrating Laravel security practices with Trivy vulnerability scanning integrated into GitHub Actions.

## Documentation

- [Slides](https://drive.google.com/file/d/1pdKXfCcVs34FIGJHDbNzuYiWV-H3f3fm/view?usp=sharing) - Presentation slides for this project

## Security Scanning

This project includes automated Trivy security scans configured in GitHub Actions workflows to detect vulnerabilities in the codebase and dependencies.
